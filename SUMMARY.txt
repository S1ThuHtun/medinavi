================================================================================
  WHAT THE USER NEEDS TO DO
================================================================================

OPTION 1: Automated Setup (Recommended - 15 minutes)
┌────────────────────────────────────────────────────────────────────┐
│ 1. Run: ./setup_firebase.sh (macOS/Linux)                          │
│    Or:  setup_firebase.bat (Windows)                               │
│                                                                     │
│ 2. Follow the prompts to:                                          │
│    - Install Firebase CLI and FlutterFire CLI                      │
│    - Login to Firebase                                             │
│    - Configure Firebase project                                    │
│    - Enable Email/Password authentication                          │
│                                                                     │
│ 3. Run: flutter run                                                │
│                                                                     │
│ 4. Test sign up and sign in                                        │
└────────────────────────────────────────────────────────────────────┘

OPTION 2: Manual Setup (20 minutes)
┌────────────────────────────────────────────────────────────────────┐
│ 1. Install tools:                                                  │
│    npm install -g firebase-tools                                   │
│    firebase login                                                  │
│    dart pub global activate flutterfire_cli                        │
│                                                                     │
│ 2. Configure Firebase:                                             │
│    flutterfire configure                                           │
│                                                                     │
│ 3. Enable Auth in Firebase Console:                                │
│    - Go to console.firebase.google.com                             │
│    - Authentication → Sign-in method                               │
│    - Enable Email/Password                                         │
│                                                                     │
│ 4. Run app:                                                        │
│    flutter pub get                                                 │
│    flutter run                                                     │
└────────────────────────────────────────────────────────────────────┘
